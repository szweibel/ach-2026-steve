# ACH 2026 — What the lab builds and runs

Reveal.js slides for Stephen Zweibel's ~10-minute section of the ACH 2026 panel
**"Building Community-Oriented Infrastructures for AI Experimentation"** (CUNY AI Lab),
Friday, June 26, 2026.

Covers the Tools Portal and agentic coding studios, Kale Workbench (governed
coding-agent work), and Kale Deploy, framed as what the lab builds and runs.

## Use

Open `index.html` in a browser. Arrow keys / Space to advance, **S** for
presenter view (speaker notes), **F** for fullscreen, **Esc** for the overview.

No build step — reveal.js loads from CDN. Files: `index.html`, `styles.css`.
